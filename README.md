# DIY Launcher

これは自由にUIをレイアウトしてコマンドを設定できる、あなたただけのランチャーが作れるアプリケーションです。

# 使いかた
1. resources フォルダの中にある config.xml をメモ帳などで開きます。
2. ボタンのタイトルやコマンド、ウィンドウサイズを自由に書き換えて保存してください。
3. DiyLauncher.exe を起動すると、設定が反映されます。

# 注意事項
- config.xml の記述を間違えると起動しない場合があります。その場合は記述を見直してください。
- XSSやコマンドインジェクション対策は行っていません。ほかの人が作成したconfig.xmlを使う場合は十分注意してください。
